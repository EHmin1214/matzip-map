# 디자인 인수인계서 — #6 인증 화면 수정

---

## 디자인 방향

카카오 로그인이 현재 비활성 상태이므로 숨기고, 로그인/회원가입 전환을 더 가볍게 정리합니다.

---

## 수정 1: 카카오 로그인 영역 숨김

**파일:** `frontend/src/components/AuthScreen.jsx`

**변경 사항:**

다음 영역을 **렌더링하지 않도록 숨김** (삭제가 아닌 주석 처리 또는 조건부 렌더링 — 나중에 복원 가능하게):

- "또는" 구분선 (212-217행)
- 카카오 로그인 버튼 (219-236행)

---

## 수정 2: 모드 전환(로그인 ↔ 회원가입) 영역 정리

**파일:** `frontend/src/components/AuthScreen.jsx` — 238-260행

**현재:**
```
처음이신가요? [계정 만들기]     (밑줄 텍스트 버튼)
```
- `borderTop` 구분선 있음
- 문장 + 밑줄 링크 조합

**변경:**

카카오 버튼이 사라지면서 제출 버튼 아래에 공간이 생기므로, 모드 전환 영역은 현재 스타일 그대로 유지하되 구분선(`borderTop`)을 **제거**합니다.

- `borderTop: \`1px solid ${C.outline}22\`` → 제거
- `paddingTop: 28` → `paddingTop: 0`으로 변경 (구분선 없으니 패딩도 불필요)
- `marginTop: 28` → **유지** (제출 버튼과의 간격)
- 텍스트 스타일 유지: "처음이신가요? **계정 만들기**" (밑줄 + bold)

**결과 레이아웃:**

```
나의 공간
─── (짧은 장식선)

다시 오셨군요
닉네임과 PIN으로 나의 공간을 열어보세요

[닉네임 입력]
[PIN 입력]

[ 시작하기 ]    ← solid 버튼

처음이신가요? 계정 만들기    ← 텍스트 링크 (밑줄)
```

카카오 영역이 빠지면서 화면이 더 가벼워지고, "나만의 공간" 느낌에 맞는 간결한 인증 화면이 됩니다.

---

## 수정하지 않는 것

- 브랜드 영역("나의 공간") 유지
- 헤더 텍스트("다시 오셨군요" / "처음 오셨군요") 유지
- 입력 필드 스타일 유지
- 제출 버튼 스타일 유지
- 에러 메시지 스타일 유지
- 데스크톱 좌측 비주얼 패널 유지
- 카카오 관련 코드는 삭제하지 말고 숨기기만 (복원 가능하게)
