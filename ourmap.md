# "우리의 공간" — 공유 큐레이션 맵 기획서

---

## 기획 의도

개인이 경험한 최고의 장소만 제한된 수로 공유하여, 바이럴 리뷰나 광고에 영향받지 않는 **순수 경험 기반 지도 데이터**를 만드는 기능입니다.

- "나의 공간" = 개인 아카이브 (모든 기록)
- "우리의 공간" = 전체 유저의 큐레이션 (베스트만)
- 개개인의 소중한 경험이 쌓여 집단의 지혜가 되는 구조
- 제한된 슬롯(카테고리당 5개)이 진정성 필터 역할

---

## 핵심 스펙

**슬롯:** 카테고리당 **5개**

**카테고리 (초기 고정형):**
- 🍽 음식점
- ☕ 카페
- 🍸 바
- 👔 옷가게

추후 사용자 수와 데이터가 쌓이면 카테고리를 세분화할 수 있음 (예: 음식점 → 한식/일식/양식)

**마커 표시:** 기존 팔로잉 마커 스타일과 유사하되, 이니셜 아바타 대신 **숫자 뱃지** (N명이 선택)
- 예: `(7) 블루보틀 성수`
- 숫자가 클수록 많은 사람이 독립적으로 "베스트"로 선택했다는 의미

---

## 레이아웃: 모바일

### 지도 상단 모드 전환

기존 지도 상단 영역을 모드에 따라 전환합니다. 레이아웃 구조는 변하지 않고 내용만 바뀝니다.

**첫 번째 줄 — 모드 토글:**
```
(나의 공간) · 우리의 공간
```
- 작고 조용한 텍스트 토글
- 활성 모드: `fontWeight: 700`, `color: C.primary (#655d54)`
- 비활성 모드: `fontWeight: 400`, `color: C.outlineVariant`
- 가운데 `·` 구분자
- 위치: 지도 상단 좌측, 기존 팔로잉 필터 위

**두 번째 줄 — 컨텍스트 필터 (가로 스크롤):**

나의 공간 모드:
```
👤민지  👤서윤  👤지호  →     (기존 팔로잉 유저 필터)
```

우리의 공간 모드:
```
전체  🍽음식점  ☕카페  🍸바  👔옷가게  →     (카테고리 필터)
```

- 기존 팔로잉 필터와 동일한 가로 스크롤 pill 스타일
- 활성 카테고리: `background: C.primary`, `color: #fff6ef`
- 비활성 카테고리: `background: C.surfaceLow`, `color: C.onSurfaceVariant`
- "전체" 옵션: 모든 카테고리의 장소를 한 번에 표시

## 레이아웃: 데스크톱

### 사이드바 모드 전환

사이드바 최상단에 모드 전환을 배치합니다. 모드에 따라 사이드바 내용 전체가 바뀝니다.

**사이드바 상단 — 모드 전환:**
```
나의 공간        ← 활성 시 bold + primary 컬러
우리의 공간       ← 비활성 시 regular + outlineVariant 컬러
```
- 세로로 두 줄 배치
- 클릭 시 전환
- 전환 시 아래 콘텐츠가 부드럽게 교체 (fade, 0.15s)

**나의 공간 모드 사이드바 (기존 그대로):**
```
(나의 공간)  ← bold
 우리의 공간
────────────
🔍 검색
📍 장소 목록
  · 스시오마카세
  · 블루보틀
  · ...
▶ 팔로잉 (3)   ← 아코디언 (인수인계서 #1 참고)
────────────
👤 프로필
```

**우리의 공간 모드 사이드바:**
```
 나의 공간
(우리의 공간)  ← bold
────────────
카테고리
  · 🍽 음식점    (활성 시 primary 배경 하이라이트)
  · ☕ 카페
  · 🍸 바
  · 👔 옷가게
────────────
인기 장소
  (7) 블루보틀 성수
  (5) 카페온더플랜
  (3) 스시오마카세
  (3) 리틀넥 한남
  ...
────────────
👤 프로필
```

**인기 장소 리스트:**
- 선택된 카테고리 기준, 선택 수(뱃지 숫자) 내림차순 정렬
- 숫자 뱃지: `fontWeight: 800`, `color: C.primary`, 괄호로 감쌈
- 장소명: `fontFamily: FL (Manrope)`, 일반 텍스트
- 클릭 시 → 지도에서 해당 위치로 이동 + RestaurantPanel 열림 (기존 패널 재활용)
- 카테고리 "전체" 선택 시 모든 카테고리 혼합 리스트

## "나의 베스트" 편집 UI

두 가지 진입점으로 구성합니다. "추가"는 장소를 보는 흐름 안에서, "관리"는 프로필에서.

### 진입점 1: RestaurantPanel에서 바로 추가 (주 동선)

**파일:** `frontend/src/components/RestaurantPanel.jsx`

장소 상세 패널의 지도/공유 버튼 영역에 **"베스트에 추가"** 버튼 추가:

```
스시오마카세
📍 서울 강남구...
🔖 가고싶다 · ⭐ 4.5

[ 지도 ]  [ 공유 ]  [ ⭐ 베스트에 추가 ]
```

**동작:**
- 클릭 시 카테고리 선택 드롭다운 (음식점/카페/바/옷가게) — 기존 지도/공유 드롭다운과 동일한 UI 패턴
- 카테고리 선택 → 해당 카테고리 베스트에 추가
- 해당 카테고리 슬롯이 가득 찬 경우 → "슬롯이 가득 찼어요" + 기존 5개 목록 표시 → 교체할 장소 선택 → 교체
- 이미 베스트에 등록된 장소 → 버튼이 "베스트 등록됨 ✓"로 변경 (비활성 스타일)
- **내 장소에만 노출** (다른 사람의 장소에는 이 버튼을 표시하지 않음)

**스타일:**
- 기존 "지도"/"공유" 버튼과 동일한 크기/패턴
- 아이콘: `star` (Material Symbols)
- 비등록 상태: `background: C.surfaceLow`, `color: C.onSurfaceVariant`
- 등록 완료 상태: `background: C.primaryContainer`, `color: C.primary`

### 진입점 2: 프로필에서 전체 관리 (보조 동선)

**파일:** `frontend/src/components/ProfilePage.jsx` 내 새 섹션

**레이아웃:**
```
나의 베스트
────────────
🍽 음식점
  1. 스시오마카세         ✕
  2. 리틀넥 한남          ✕
  3. ┈ ┈ ┈ 채워보세요 ┈ ┈ ┈
  4. ┈ ┈ ┈ 채워보세요 ┈ ┈ ┈
  5. ┈ ┈ ┈ 채워보세요 ┈ ┈ ┈

☕ 카페
  1. ┈ ┈ ┈ 채워보세요 ┈ ┈ ┈
  ...
```

**빈 슬롯:**
- 점선 테두리 (`border: 1px dashed C.outline`)
- "채워보세요" 텍스트: `color: C.outlineVariant`, `fontStyle: italic`
- 클릭 시 → 내 저장 장소 중에서 선택하는 피커 모달

**채워진 슬롯:**
- 장소명 표시
- 우측 ✕ 버튼으로 제거 가능
- 드래그로 순서 변경 (선택적, 구현 부담이 크면 생략)

**선택 피커 모달:**
- 내가 "나의 공간"에 저장한 장소 목록에서 선택
- 해당 카테고리에 맞는 장소만 필터링 (가능한 경우)
- 이미 베스트에 등록된 장소는 비활성 표시
- 검색 가능

**의도:** 대부분의 추가는 진입점 1(RestaurantPanel)에서 자연스럽게 이뤄지고, 프로필은 전체 현황 확인 및 제거/교체용

## 마커 스타일: 우리의 공간

**파일:** `frontend/src/components/MapView.jsx`에 새 마커 함수 추가

**pill 마커 (줌 12 이상):**
```
┌──────────────────┐
│ (7) 블루보틀 성수  │
└──────────────────┘
```
- `background: rgba(255,255,255,0.92)` (흰색 반투명)
- `border: 1.5px solid C.primary (#655d54)`
- `color: C.primary`
- 숫자 뱃지 `(7)`: `fontWeight: 800`
- 장소명: `fontWeight: 600`

**dot 마커 (줌 12 미만):**
- 기존 dot과 동일 형태
- `background: C.primary (#655d54)`
- 크기: 8px

**의도:** 우리의 공간 마커는 개인 소유가 아니므로 특정 유저 컬러가 아닌 **서비스 primary 컬러** 통일. outline 스타일로 내 마커(나의 공간 모드에서 볼 때)와 구분.

## 데이터/API (백엔드 참고)

**필요한 엔드포인트 (예상):**

```
GET  /shared-map/?category=restaurant          → 카테고리별 장소 목록 (선택 수 포함)
GET  /shared-map/my-picks?user_id=...          → 내 베스트 목록
POST /shared-map/my-picks?user_id=...          → 베스트에 장소 추가 { place_id, category }
DELETE /shared-map/my-picks/{pick_id}?user_id=... → 베스트에서 장소 제거
```

**응답 형태 (예상):**
```json
{
  "place_id": 123,
  "name": "블루보틀 성수",
  "address": "서울 성동구...",
  "category": "cafe",
  "pick_count": 7,
  "lat": 37.5445,
  "lng": 127.0567
}
```

## 기존 코드와의 연결

- **MapView.jsx:** 모드 상태 추가 (`mapMode: "personal" | "shared"`), 모드에 따라 다른 마커 세트 렌더
- **App.js:** `mapMode` 상태 관리, 사이드바/모바일 컨트롤에 전달
- **Sidebar.jsx:** 상단에 모드 전환 UI 추가, 모드에 따라 내용 교체
- **MobileMapControls.jsx:** 상단 토글 + 카테고리 필터 추가
- **ProfilePage.jsx:** "나의 베스트" 섹션 추가
- **RestaurantPanel.jsx:** 우리의 공간 장소 클릭 시에도 기존 패널 재활용 (선택 수 뱃지 추가 표시)
